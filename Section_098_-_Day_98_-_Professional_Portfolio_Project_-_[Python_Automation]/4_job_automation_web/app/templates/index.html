Welcome to Your Job Automation Dashboard

This project is a template for automating repetitive work tasks – like filling web forms, generating reports, or sending emails.

The data you see (names, emails, departments, scores) is just example data to show how everything fits together. In your real job, you would replace it with your own data and customize the automation script to perform the task you need.

What does the automation actually do?

Right now, the automation engine does:

  • Reads each row from your input CSV.
  • If web automation is enabled in config.yaml, it logs into a website and fills a form with the data from each row.
  • After processing all rows, it generates a summary report (statistics of numeric columns like "Score") and saves it to data/output_reports/report.csv.
  • If email is enabled, it sends that report to a recipient.

The "Score" column is just a numeric field to demonstrate statistics – in your real data it could be anything (e.g., revenue, quantity, rating).

Quick Start – Make It Your Own
  1. Replace the example data in data/input_data/data.csv with your own data.
  2. Configure the automation in config.yaml:
    • Set web.enabled to true if you need web forms.
    • Update login URL, form field IDs, and mapping.
    • Enable email if desired.
  3. Customize the automation logic in app/services/automation_engine.py:
    • Modify fill_form() to interact with your actual web form.
  4. Start the automation from the Automation page.
