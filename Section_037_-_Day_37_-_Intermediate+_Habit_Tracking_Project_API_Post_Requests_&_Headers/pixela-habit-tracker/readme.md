To run this project, you need to create a `.env` file in the root directory with the following variables:

```env
PIXELA_USERNAME=your_username
PIXELA_TOKEN=your_token
PIXELA_GRAPH_ID=your_graph_id
PIXELA_BASE_URL=https://pixe.la/v1

```

install dependencies with `pip install -r requirements.txt`

run with `python -B main.py`
